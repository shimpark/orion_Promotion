- 버튼 input

  • 버튼이 함께 있는 input 입니다.
  • 버튼의 사이즈에 따라 small / medium / large 로 구분할 수 있습니다.

- 달력 input

  • 달력이 포함되어있는 input 입니다.
  • 달력은 년 / 월 / 시간 달력으로 구분되어 있습니다.
  • .month클래스를 추가하면 월 달력이 됩니다.
  • .time클래스를 추가하면 시간 달력이 됩니다.
